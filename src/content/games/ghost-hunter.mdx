### Overview

I adapted a Favicon Game Jam entry for A Game By Its Cover Jam 2023. The original concept is by game artist [Gabby DaRienzo](//gabbydarienzo.com/). Loved the colors and the idea of having ghost friends.

The code is up on [GitHub](//github.com/cmoles/ghost-hunter). Will add to this page as I make updates to the game.
